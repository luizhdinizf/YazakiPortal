 Vídeo #4:
CNC 3.0 | Parte Eletrônica/Drive A4988 e Ajuste de Corrente/CNC Shield 3.0/GRBL 0.9J/Motores

Baixar o GRBL:

 Caso você prefira já fiz a alteração na linha //#define VARIABLE_SPINDLE // Default enabled. Comment to disable. e deixei a pasta pronta para você só instalar na IDE do arduino. Ou seja, esta pasta contem o GRBL sem o controle PWM do spindle, só adiciona-lá as bibliotecas da IDE do Arduino.

 Clique aqui para baixar a pasta já pronta

 Caso prefira fazer todo o processo segue os links abaixo:

 Para baixar o grbl master sem alteração: 
https://github.com/grbl/grbl/

Para visualizar os pinos do GRBL no arduino:
https://github.com/gnea/grbl/wiki/Connecting-Grbl

1° - Linha a ser localizada. Abrar a pasta grbl-master, grbl,  abra o arquivo config com o bloco de notas e pressione Ctrl + F e cole a linha abaixo:
#define VARIABLE_SPINDLE // Default enabled. Comment to disable.
2° - O bloco de notas irá localizar a linha, depois insira o // no começo da linha, fica assim:
//#define VARIABLE_SPINDLE // Default enabled. Comment to disable.


 3 ° - Depois salve, e instale a biblioteca na IDE do Arduino, ou baixe a primeira pasta pronta e fique tranquilo.
